Moody

/ ˈmo͞odē /

Web Prototype and Wireframes

Moody was a design project done within the UEGroup to replace the current system of recording Emotional Chart data with a simple to use web application. The philosophy behind the project is to create a web application that is as easy or easier to use than the current system. We will accomplish this task by improving the overall efficiency, reducing the amount of data entry and reducing the margin for error.

Phase 1: Empathize and Web Prototype

•  Empathized self with the UEGroup’s Emotional Chart by shadowing tool being used and interviewing researchers
•  Created web prototype of UEGroup’s Emotional Chart to see if project was feasible to create. Web prototype built using HTML / CSS / JS / Node.js / Express.js
