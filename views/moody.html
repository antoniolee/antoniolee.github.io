Moody

/ ˈmo͞odē /

Web Prototype and Wireframes

Moody was a design project done within the UEGroup to replace their current system of recording research participants "emotional level" with a simple to use web application. The philosophy behind the project was to create a web application that is as easy or easier to use than the current system of recording data by hand. This design was accomplished by improving the overall efficiency of recording data, reducing the amount of data entry and reducing the margin for error.

Phase 1: Empathize and Web Prototype

•  Shadowed UEGroup Researcher's using the current system
•  Created web prototype of UEGroup’s Emotional Chart system. Web prototype built using HTML, CSS, JS, and Express.js
